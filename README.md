# sORFplnc
A bioinformatics tool to effectively find small open reading frames in plant lncRNA

# OS
win10

# Dependencies
Language dependency: Python 3 (Please do not use Python 2 to run the code.)

# Library dependency

python==3.6.15

numpy==1.19.2

scikit_learn==0.24.2

# Usage

python sORFplnc.py

Also your data your way.
